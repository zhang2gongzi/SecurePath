# SecureProtoCoT 实验执行清单

> **目标**：基于MSR_data_cleaned数据集训练安全感知编码器，应用于CoT推理路径安全选择
> **创建日期**：2026-05-18
> **最后更新**：2026-05-19

---

## 一、技术路线概览

```
阶段一：安全感知编码器训练                阶段二：CoT路径安全选择
┌─────────────────────────┐           ┌─────────────────────────┐
│  MSR_data_cleaned       │           │  LLM多路径采样          │
│  (188,636函数)          │           │  (K条推理路径)          │
│         ↓               │           │         ↓               │
│  数据预处理             │           │  路径编码               │
│  (清洗/划分/标注)       │           │  (使用训练好的编码器)   │
│         ↓               │           │         ↓               │
│  对比学习训练           │ ────────→ │  安全得分计算           │
│  (漏洞代码 vs 修复代码) │  编码器   │  (与安全原型相似度)     │
│         ↓               │           │         ↓               │
│  安全感知编码器         │           │  安全路径选择           │
└─────────────────────────┘           └─────────────────────────┘
```

## 二、数据集信息

### 2.1 MSR_data_cleaned 核心统计

| 指标 | 数值 |
|------|------|
| 总函数数 | 188,636 |
| 漏洞函数 (vul=1) | 10,900 |
| 非漏洞函数 (vul=0) | 177,736 |
| 文件大小 | ~10GB |

### 2.2 内存安全漏洞分布（6种CWE类型）

**原始数据分布**：

| CWE | 漏洞类型 | 原始数量 | 占比 |
|-----|----------|----------|------|
| CWE-119 | 缓冲区溢出 | 26,497 | 48.7% |
| CWE-416 | Use After Free | 9,780 | 18.0% |
| CWE-125 | 越界读取 | 7,717 | 14.2% |
| CWE-476 | 空指针解引用 | 4,572 | 8.4% |
| CWE-190 | 整数溢出 | 3,671 | 6.7% |
| CWE-787 | 越界写入 | 2,918 | 5.4% |

**实验采样策略**：

| CWE | 漏洞类型 | 采样数量 | 用途 |
|-----|----------|----------|------|
| CWE-119 | 缓冲区溢出 | 1,000 | 训练/测试 |
| CWE-416 | Use After Free | 1,000 | 训练/测试 |
| CWE-125 | 越界读取 | 1,000 | 训练/测试 |
| CWE-476 | 空指针解引用 | 1,000 | 训练/测试 |
| CWE-190 | 整数溢出 | 1,000 | 训练/测试 |
| CWE-787 | 越界写入 | 1,000 | 训练/测试 |
| **总计** | | **6,000** | |

**说明**：每种CWE类型均匀采样1000条，确保类别平衡，加速训练和实验迭代。

### 2.3 关键字段

- `func_before` - 漏洞代码（正样本）
- `func_after` - 修复代码（负样本）
- `lines_before` - 漏洞行
- `lines_after` - 修复行
- `vul` - 是否为漏洞函数（1/0）
- `CWE ID` - 漏洞类型
- `project` - 项目名称

## 三、分阶段任务清单

### 阶段1：数据预处理（预计2-3天）

#### 1.1 数据加载与清洗
| 任务 | 具体内容 | 输出 | 状态 |
|------|----------|------|------|
| 加载原始数据 | `pd.read_csv('MSR_data_cleaned.csv')` | DataFrame | ⬜ 待开始 |
| 过滤有效样本 | func_before和func_after非空 | 清洗后数据 | ⬜ 待开始 |
| 去除异常数据 | 过短(<10行)和过长(>200行)函数 | 有效样本集 | ⬜ 待开始 |
| 按CWE分组 | 按6种漏洞类型分组 | 分组数据 | ⬜ 待开始 |

#### 1.2 数据划分策略

**策略一：按比例划分（RQ3主实验）**

| 数据集 | 比例 | 各CWE数量 | 总计 |
|--------|------|-----------|------|
| 训练集 | 70% | 700/种 | 4,200 |
| 验证集 | 10% | 100/种 | 600 |
| 测试集 | 20% | 200/种 | 1,200 |

**策略二：按CWE类型划分（RQ4泛化实验）**

| 划分 | CWE类型 | 数量 | 用途 |
|------|---------|------|------|
| 训练集 | CWE-119/416/125/476/190 | 5,000 | 训练编码器 |
| 测试集 | CWE-787（留一） | 1,000 | 测试泛化能力 |

**实验设计**：

| 实验 | 划分策略 | 目的 |
|------|----------|------|
| RQ3 漏洞检测效果 | 策略一（按比例） | 验证同类漏洞检测能力 |
| RQ4 跨类型泛化 | 策略二（留一法） | 验证未见CWE类型的泛化 |

**产出**：
- `data/processed/train_ratio.csv` (策略一训练集)
- `data/processed/val_ratio.csv` (策略一验证集)
- `data/processed/test_ratio.csv` (策略一测试集)
- `data/processed/train_leave_one.csv` (策略二训练集)
- `data/processed/test_leave_one.csv` (策略二测试集)

#### 1.3 标签构建
| 样本类型 | 来源 | 标签 |
|----------|------|------|
| 漏洞代码 | func_before (vul=1) | 1 (危险) |
| 安全代码 | func_after | 0 (安全) |

**产出**：
- `data/processed/train.csv`
- `data/processed/val.csv`
- `data/processed/test.csv`
- `data/processed/cwe_stats.json`

## 五、评估指标

### 5.1 漏洞检测指标
| 指标 | 定义 | 计算方式 |
|------|------|----------|
| Accuracy | 分类准确率 | (TP+TN)/(TP+TN+FP+FN) |
| Precision | 精确率 | TP/(TP+FP) |
| Recall | 召回率 | TP/(TP+FN) |
| F1 | F1分数 | 2*P*R/(P+R) |
| AUC | ROC曲线下面积 | sklearn计算 |

### 5.2 表示学习指标
| 指标 | 定义 | 计算方式 |
|------|------|----------|
| Silhouette Score | 聚类质量 | sklearn.metrics.silhouette_score |
| Davies-Bouldin | 聚类紧密度 | sklearn.metrics.davies_bouldin_score |

### 5.3 CoT路径选择指标
| 指标 | 定义 | 计算方式 |
|------|------|----------|
| 安全选择率 | 选择安全路径的比例 | 安全选择数/总选择数 |
| 漏洞规避率 | 成功规避漏洞的比例 | 1 - 危险选择数/总危险路径 |
